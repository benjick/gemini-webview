# Gemini Webview

Wraps Gemini in a webview.

![screenshot](.github/app.png)

## Installation

1. Download the latest release from [GitHub releases](https://github.com/benjick/gemini-webview/releases).
2. Open the downloaded file.
3. Follow the instructions at https://support.apple.com/guide/mac-help/open-a-mac-app-from-an-unknown-developer-mh40616/mac

## Development

```bash
pnpm install
pnpm dev
```

## Building locally

```bash
pnpm install
pnpm build
```
